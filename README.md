## Structure du Projet

```
Scripts/
├── src/
│   ├── main/
│   │   └── StationLavageMain.java (package: main)
│   └── model/
│       ├── Client.java (package: model)
│       ├── RendezVous.java (package: model)
│       └── Prestation/
│           ├── Prestation.java (package: model.Prestation)
│           ├── PrestationExpress.java
│           ├── PrestationSale.java
│           └── PrestationTresSale.java
└── nbproject/
```

## Démarrage Rapide

1. Cloner le repository
2. **Ouvrir le dossier `Scripts/` dans NetBeans** (pas la racine !)
3. Clean and Build
4. Exécuter `main.StationLavageMain`

Voir [SETUP.md](SETUP.md) pour plus de détails.
